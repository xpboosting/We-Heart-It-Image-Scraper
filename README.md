# WeHeartIt Scraper
Simple Node.JS scraper for [weheartit](https://weheartit.com/) 

## Installation

Use [git](https://git-scm.com/) to clone the package

```bash
git clone https://github.com/NotII/whiScraper/
cd whiScraper
npm i
```

## Usage

Edit the config.json file and put in a canary webhook
```
node index.js
```
